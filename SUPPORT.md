# Support

## Getting Help

- Open a GitHub issue for bugs and technical questions.
- Include logs, config snippets (sanitized), and exact commands where possible.

## What to Include

- Quaid version
- Host/runtime (OpenClaw, Claude Code, Codex, standalone CLI)
- OS and environment details
- Steps to reproduce
- Expected vs actual behavior

## Known Reality (Alpha)

Quaid is in early alpha. OpenClaw is the most mature path. Claude Code and Codex are active integrations, but some host-specific behaviors remain less validated than the OpenClaw reference path.
